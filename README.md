# Timey

A Pomodoro timer app built with Vue 3 and TypeScript.

Timey is being built step by step with a logic-first approach:

1. Define timer domain and state clearly
2. Build and test timer engine behavior
3. Add persistence and notifications
4. Polish UI/UX after logic is stable

## Tech Stack

- Vue 3
- TypeScript
- Vite

## Planned Features

- Focus / short break / long break modes
- Start / pause / resume / reset / skip controls
- Auto mode transitions
- Configurable durations
- Local storage persistence
- Optional browser notifications
- Accessibility-first controls
